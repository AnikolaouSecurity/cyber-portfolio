# BSc Thesis: Decoding Reed-Solomon Codes

## Overview
Analysis and breakdown of Shuhong Gao's algorithm for decoding Reed-Solomon codes, a fundamental error-correcting code used in digital communications and storage systems.

## Key Topics
- **Reed-Solomon Codes** — error-correcting codes based on polynomial interpolation over finite fields
- **Shuhong Gao's Algorithm** — an efficient decoding approach using Gröbner basis techniques
- **Abstract Algebra foundations** — finite fields, polynomial rings, group theory applied to coding theory
